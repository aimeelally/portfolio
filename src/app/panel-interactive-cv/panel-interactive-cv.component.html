Side Scrolling Experiment

This is a challenge I've set for myself to build a side scrolling parallax site using html, css(scss), javascript and jquery (wrapped in Angular1). It's buggy but it's getting there.

Even though this is still under construction, I am making it available for anyone who wants to poke into the code to see how it is held together so far.

If you're using a mobile/tablet, you may spot some issues, but fret not, I have a list of bugs as long as my arm which should be ironed out over the next few weeks.

Feel free to log any bugs with me via the contact form (please include device specs) and I shall add it to my very long list.
