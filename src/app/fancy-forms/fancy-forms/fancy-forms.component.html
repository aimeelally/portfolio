Fancy Forms

Native form elements are sometimes limited in their styling capabilities (I'm looking at you select boxes!). I have come across instances where some major companies (who shall remain nameless but you would totally know them!) have even tried to mock the behaviour of a select box using javascript instead of using the native element. This creates a whole host of issues when it comes to Accessibility, Keyboard Shortcuts and Default Browser Behaviour.

To combat this, with the help of my old team in Adaptemy, we created a selection of fully accessible native form elements available via an NPM module if you'd like to try them out in your own project. All design credit goes to Maurice Glennon.

If you have questions about the module or spot any issues, be sure to send me a message via the contact form.
